独立验证抽奖结果

不信任平台，自己动手验证。三步完成。

1

获取收盘价

从公开渠道获取开奖日的上证指数和深证成指收盘价，独立计算 seed

2

获取链上参数

在区块链浏览器中查看 drawWinner 交易的 input data，提取 seed、blockNumber、blockTs

3

重算比对

用相同算法重新计算中奖结果，与链上记录比对，完全一致即为公正

Seed 计算公式

上证收盘指数 × 深证收盘指数 × 10000 = 12位整数 → 倒序排列 → seed

示例：
  上证 3200.15，深证 10800.50
  → 3200.15 × 10800.50 × 10000
  → 取整数部分（12位）
  → 倒序
  → seed

链上验证算法

// 第一个中奖者
uint256(keccak256(abi.encodePacked(blockTs, blockNumber, seed))) % 参与人数

// 后续中奖者
uint256(keccak256(abi.encodePacked(前一中奖索引, blockTs, blockNumber, seed))) % 参与人数

// 如果抽到已中奖者，自动顺延至下一个
